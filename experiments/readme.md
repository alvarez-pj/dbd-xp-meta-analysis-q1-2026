# Experiments Hub

This folder hosts the **experiment list** for the DbD meta-analysis: 61 experiments (XPs) that launched on DoubleDash between 2023 and 2026.

The list includes:
- **Context** — links to PRDs, briefs, and readouts
- **Launch date**
- **Surface** — Post-Checkout, Pre-Checkout, or Both
- **Curie links** — for metric results

The master list is `experiment_list.csv`. Use the **analysis CSV** column to link each row to its result CSV in `analysis/` for merging and meta-analysis.
